

## Build

### Clone 
```
git clone https://github.com/gyetis/planto3d
```
```
cd /path/to/planto3d
```

### Conda


```
conda env create -f environment.yml
```
```
chmod +x run.sh
```
### Docker
```
docker build -t planto3d .
```
```
chmod +x run_docker.sh
```

## Run
### Conda
```
./run.sh
```
### Docker
```
./run_docker.sh
```
