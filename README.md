# Planto3D

### (Under development) Web app for automated 3D housing model generation from architectural drawings using ML

## Build

```
git clone https://github.com/gyetis/planto3d
```
```
cd /path/to/planto3d
```

```
conda env create -f environment.yml
```
```
chmod +x run.sh
```

## Run
### Conda
```
./run.sh
```
